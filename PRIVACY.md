Полная версия с деталями по слоям хранения и правами — [docs/DATA-POLICY.md](docs/DATA-POLICY.md) ([DP.D.035](https://github.com/TserenTserenov/PACK-digital-platform/blob/main/pack/digital-platform/01-domain-contract/DP.D.035-data-policy.md) для технических деталей).

## Что собирается

Объём зависит от тира использования — от «ничего» (без аккаунта) до полного цифрового двойника (профиль, цели, самооценка, git-коммиты) на верхнем тире. Таблица по тирам — [docs/DATA-POLICY.md §1](docs/DATA-POLICY.md#1-какие-данные-собираются).

## Куда уходят данные (opt-in / внешние сервисы)

| Сервис | Что отправляется | Обязательность |
|---|---|---|
| **Anthropic API** (Claude) | Промпт + профиль + текст вопроса | Обязательно для работы агента |
| **GitHub** | Содержимое ваших репозиториев | При `push` — под вашим контролем |
| **WakaTime** | Метаданные (проект, язык, время), без содержимого файлов | Опционально, отключаемо |
| **MCP iwe-knowledge (Gateway)** | Текст поисковых запросов, без PII | Только при использовании поиска |

Anthropic не использует данные API для обучения моделей ([API Terms](https://www.anthropic.com/api-terms)).

## Что не покидает вашу машину

`memory/`, `extensions/`, `params.yaml` и весь контекст DS-репозиториев существуют только на вашем компьютере и в вашем приватном GitHub — этот шаблон-репозиторий (`FMT-exocortex-template`) к ним доступа не имеет.

## Ваши права

- **Просмотр** — команда `/mydata` в боте
- **Удаление** — `Удалить. {Ваше имя}` в боте, каскадно по всем таблицам сервера; локальные файлы и приватный репозиторий остаются под вашим контролем
- **Экспорт** — `git clone` в любой момент, данные хранятся в открытых форматах (Git, Markdown, YAML)
- **Отзыв согласия** — отключение WakaTime, отзыв GitHub OAuth, удаление локальных файлов
